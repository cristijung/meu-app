# React Native com Expo router
Comandi para instalação: `npx create-expo-app meu-app -t tabs`

## Requisitos:
- Criar um App com no mínimo 3 telas (3 rotas) - Assunto: livre
- TypeScript
- Expo Router
- Styled Component
- Splash Screen
- Modal
- Form (login e senha)
- Imagens
- Vídeos (opcional)
- 2 testes de tela
